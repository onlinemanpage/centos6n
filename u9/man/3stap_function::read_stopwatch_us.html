NAME function::read_stopwatch_us - Reads the time in microseconds for a stopwatch

SYNOPSIS read_stopwatch_us:long(name:string)

ARGUMENTS name stopwatch name

DESCRIPTION Returns time in microseconds for stopwatch name. Creates stopwatch name if it does not currently exist.
