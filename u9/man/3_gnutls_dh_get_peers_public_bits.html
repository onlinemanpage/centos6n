NAME gnutls_dh_get_peers_public_bits - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_dh_get_peers_public_bits(gnutls_session_t session);

ARGUMENTS gnutls_session_t session is a gnutls session

DESCRIPTION Get the Diffie-Hellman public key bit size. Can be used for both anonymous and ephemeral Diffie-Hellman.

RETURNS the public key bit size used in the last Diffie-Hellman key exchange with the peer, or a negative value in case of error.
