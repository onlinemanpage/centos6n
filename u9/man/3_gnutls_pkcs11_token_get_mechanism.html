NAME gnutls_pkcs11_token_get_mechanism - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pkcs11_token_get_mechanism(const char * url, int idx, unsigned long * mechanism);

ARGUMENTS const char * url should contain a PKCS 11 URL

int idx The index of the mechanism

unsigned long * mechanism The PKCS 11 mechanism ID

DESCRIPTION This function will return the names of the supported mechanisms by the token. It should be called with an increasing index until it return GNUTLS_E_REQUESTED_DATA_NOT_AVAILABLE.

RETURNS zero on success or a negative value on error.
