NAME gnutls_pkcs11_privkey_export_url - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pkcs11_privkey_export_url(gnutls_pkcs11_privkey_t key, gnutls_pkcs11_url_type_t detailed, char ** url);

ARGUMENTS gnutls_pkcs11_privkey_t key Holds the PKCS 11 key

gnutls_pkcs11_url_type_t detailed non zero if a detailed URL is required

char ** url will contain an allocated url

DESCRIPTION This function will export a URL identifying the given key.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
