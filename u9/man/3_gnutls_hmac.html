NAME gnutls_hmac - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_hmac(gnutls_hmac_hd_t handle, const void * text, size_t textlen);

ARGUMENTS gnutls_hmac_hd_t handle is a gnutls_cipher_hd_t structure.

const void * text the data to hash

size_t textlen The length of data to hash

DESCRIPTION This function will hash the given data using the algorithm specified by the context.

RETURNS Zero or a negative value on error.

SINCE 2.10.0
