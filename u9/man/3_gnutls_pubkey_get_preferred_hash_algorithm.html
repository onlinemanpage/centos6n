NAME gnutls_pubkey_get_preferred_hash_algorithm - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pubkey_get_preferred_hash_algorithm(gnutls_pubkey_t key, gnutls_digest_algorithm_t * hash, unsigned int * mand);

ARGUMENTS gnutls_pubkey_t key Holds the certificate

gnutls_digest_algorithm_t * hash The result of the call with the hash algorithm used for signature

unsigned int * mand If non zero it means that the algorithm MUST use this hash. May be NULL.

DESCRIPTION This function will read the certifcate and return the appropriate digest algorithm to use for signing with this certificate. Some cer- tificates (i.e. DSA might not be able to sign without the preferred algorithm).

RETURNS the 0 if the hash algorithm is found. A negative value is returned on error.

SINCE 2.11.0
