NAME gnutls_pubkey_import - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pubkey_import(gnutls_pubkey_t key, const gnutls_datum_t * data, gnutls_x509_crt_fmt_t format);

ARGUMENTS gnutls_pubkey_t key The structure to store the parsed public key.

const gnutls_datum_t * data The DER or PEM encoded certificate.

gnutls_x509_crt_fmt_t format One of DER or PEM

DESCRIPTION This function will convert the given DER or PEM encoded Public key to the native gnutls_pubkey_t format.The output will be stored * in @ key. If the Certificate is PEM encoded it should have a header of "PUBLIC KEY".

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
