NAME gnutls_hash_fast - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_hash_fast(gnutls_digest_algorithm_t algorithm, const void * text, size_t textlen, void * digest);

ARGUMENTS gnutls_digest_algorithm_t algorithm the hash algorithm to use

const void * text the data to hash

size_t textlen The length of data to hash

void * digest is the output value of the hash

DESCRIPTION This convenience function will hash the given data and return output on a single call.

RETURNS Zero or a negative value on error.

SINCE 2.10.0
