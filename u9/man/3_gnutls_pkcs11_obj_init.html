NAME gnutls_pkcs11_obj_init - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pkcs11_obj_init(gnutls_pkcs11_obj_t * obj);

ARGUMENTS gnutls_pkcs11_obj_t * obj The structure to be initialized

DESCRIPTION This function will initialize a pkcs11 certificate structure.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
