NAME gnutls_pkcs11_add_provider - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pkcs11_add_provider(const char * name, const char * params);

ARGUMENTS const char * name The filename of the module

const char * params should be NULL

DESCRIPTION This function will load and add a PKCS 11 module to the module list used in gnutls. After this function is called the module will be used for PKCS 11 operations.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
