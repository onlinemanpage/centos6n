NAME gnutls_global_set_time_function - API function

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_global_set_time_function(gnutls_time_func time_func);

ARGUMENTS gnutls_time_func time_func its the system time function

DESCRIPTION This is the function where you can override the default system time function.

gnutls_time_func is of the form, time_t (*gnutls_time_func)( time*);
