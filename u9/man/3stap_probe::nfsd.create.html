NAME probe::nfsd.create - NFS server creating a file(regular,dir,device,fifo) for client

SYNOPSIS nfsd.create

VALUES iap_mode file access mode

iap_valid Attribute flags

filelen the length of file name

filename file name

fh file handle (the first part is the length of the file handle)

client_ip the ip address of client

type file type(regular,dir,device,fifo ...)

DESCRIPTION Sometimes nfsd will call nfsd_create_v3 instead of this this probe point.

SEE ALSO tapset::nfsd(3stap)
