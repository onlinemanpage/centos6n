NAME gnutls_session_ticket_key_generate - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_session_ticket_key_generate(gnutls_datum_t * key);

ARGUMENTS gnutls_datum_t * key is a pointer to a gnutls_datum_t which will contain a newly created key.

DESCRIPTION Generate a random key to encrypt security parameters within Ses- sionTicket.

RETURNS On success, GNUTLS_E_SUCCESS (0) is returned, or an error code.

SINCE 2.10.0
