NAME gnutls_privkey_import_openpgp - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_privkey_import_openpgp(gnutls_privkey_t pkey, gnutls_openpgp_privkey_t key, unsigned int flags);

ARGUMENTS gnutls_privkey_t pkey The private key

gnutls_openpgp_privkey_t key The private key to be imported

unsigned int flags should be zero

DESCRIPTION This function will import the given private key to the abstract gnutls_privkey_t structure.

The gnutls_openpgp_privkey_t object must not be deallocated during the lifetime of this structure. The subkey set as preferred will be used, or the master key otherwise.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
