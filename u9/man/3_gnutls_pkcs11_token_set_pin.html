NAME gnutls_pkcs11_token_set_pin - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pkcs11_token_set_pin(const char * token_url, const char * oldpin, const char * newpin, unsigned int flags);

ARGUMENTS const char * token_url A PKCS 11 URL specifying a token

const char * oldpin old users PIN

const char * newpin new users PIN

unsigned int flags one of gnutls_pkcs11_pin_flag_t

DESCRIPTION This function will modify or set a users PIN for the given token. If it is called to set a user pin for first time the oldpin must be NULL.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
