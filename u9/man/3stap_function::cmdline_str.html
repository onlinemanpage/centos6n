NAME function::cmdline_str - Fetch all command line arguments from current process

SYNOPSIS cmdline_str:string()

ARGUMENTS None

DESCRIPTION Returns all arguments from the current process delimited by spaces. Returns the empty string when the arguments cannot be retrieved.
