NAME gnutls_x509_crq_verify - API function

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crq_verify(gnutls_x509_crq_t crq, unsigned int flags);

ARGUMENTS gnutls_x509_crq_t crq is the crq to be verified

unsigned int flags Flags that may be used to change the verification algo- rithm. Use OR of the gnutls_certificate_verify_flags enu- merations.

DESCRIPTION This function will verify self signature in the certificate request and return its status.

RETURNS On success, GNUTLS_E_SUCCESS is returned, GNUTLS_E_PK_SIG_VERIFY_FAILED if verification failed, otherwise a negative error value.
