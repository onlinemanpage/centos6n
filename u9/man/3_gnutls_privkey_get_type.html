NAME gnutls_privkey_get_type - API function

SYNOPSIS #include <gnutls/gnutls.h>

gnutls_privkey_type_t gnutls_privkey_get_type(gnutls_privkey_t key);

ARGUMENTS gnutls_privkey_t key should contain a gnutls_privkey_t structure

DESCRIPTION This function will return the type of the private key. This is actually the type of the subsystem used to set this private key.

RETURNS a member of the gnutls_privkey_type_t enumeration on success, or a neg- ative value on error.
