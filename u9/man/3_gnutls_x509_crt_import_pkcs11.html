NAME gnutls_x509_crt_import_pkcs11 - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_x509_crt_import_pkcs11(gnutls_x509_crt_t crt, gnutls_pkcs11_obj_t pkcs11_crt);

ARGUMENTS gnutls_x509_crt_t crt A certificate of type gnutls_x509_crt_t

gnutls_pkcs11_obj_t pkcs11_crt A PKCS 11 object that contains a certificate

DESCRIPTION This function will import a PKCS 11 certificate to a gnutls_x509_crt_t structure.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
